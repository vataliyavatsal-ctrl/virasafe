VIRA KYA HAI?

VIRA ek Women Safety PWA hai — ek aisa smart dashboard jo tumhe unsafe situations mein turant madad karta hai. Yeh app kaam karta hai bina kisi dedicated app install kiye, seedha browser mein.

KAISE KAAM KARTA HAI?
1
📍 Location Track
App tumhari real-time GPS location track karta hai aur MAPS tab mein live dikhata hai. Danger zones automatically map par red circles mein mark hote hain.
2
🆘 SOS Alert
SOS dabao — 5 second ka countdown milega cancel karne ke liye. Time khatam hone par saved contacts ko WhatsApp/SMS pe tumhari live location automatically jaayegi.
3
🛡️ Safe Mode
Akele travel karte waqt Safe Mode ON karo. Timer set karo — agar tum time pe check-in nahi karte toh auto-SOS trigger ho jaata hai. Long travel ke liye 8h timer bhi hai.
4
👥 Contacts
CONTACTS tab mein max 5 emergency contacts save karo (naam + 10-digit number). Yahi log SOS alert receive karenge. Jitne zyada contacts, utna better.
5
🚨 Report Unsafe Area
Koi jagah unsafe lagi? MAPS tab pe Report button dabao — tumhari location par ek Red Danger Zone ban jaayega jo sab users ko warn karega.
6
⚡ Ecosystem
ECO tab mein permissions grant karo, aur VIRA Awareness Engine se safety tips padho — stalking, raat ka safar, unknown cab — sab situations ke liye guide.

🧠 VIRA Awareness Engine
Guide
😰
Stalking / Following
Koi peeche aa raha hai — kya karo?
▶
1
Route badlo — seedha ghar mat jao, public jagah jao
2
Phone nikaalo aur call karo — signal deta hai ki tum akele nahi ho
3
Safe Mode ON karo — family automatically alert hogi
🤖 VIRA: Safe Mode Timer set karo, time pe check-in nahi hua toh auto-SOS trigger hoga.
🌑
Raat ka Safar
Unsafe route pe kaise safe rahein?
▶
1
Route share karo — Google Maps live share + VIRA Safe Mode dono ON
2
Earphones mat lagao — surroundings sunna zaruri hai
3
Crowd wale raaste lo — shortcuts avoid karo raat mein
🤖 VIRA: Safe Mode timer set karo — destination pe pahunch ke band karo.
🚖
Unknown Cab / Auto
Safar mein safe kaise rahein?
▶
1
Number plate photo lo aur family ko bhejo
2
Live location share karo — VIRA Safe Mode ON karo
3
Route galat lage — SOS dabao, 5 sec cancel window milegi
🤖 VIRA: Expected arrival time pe timer set karo — miss hone pe auto-SOS.
